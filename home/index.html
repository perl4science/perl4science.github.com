Welcome to the Perl for Science website

This page is intended to promote all those ways that we use our favorite language to get things done in all scientific disciplines. Go ahead, add links to your project, provide examples and give installation help. Just fork the website’s repo, add your content and submit a pull request.

Useful Projects

  • Perl Data Language - PDL turns Perl into a free, array-oriented, numerical language
  • BioPerl - a community effort to produce Perl code which is useful in biology
  • PerlGSL - new project to provide the Gnu Scientific Library to Perl with a Perlish feel
