Welcome

This site is intended to promote all those ways that we use our favorite language to get things done in all scientific disciplines. Go ahead, add links to your project, provide examples and give installation help. Just fork the website’s repo, add your content and submit a pull request.

The Quantified Onion

This site has a sister Google Group called The Quantified Onion. This is the place for Perl Scientists and Friends to interact. Use it to post your experiences, questions, links to publications or anything else Perl or Science related. Hope to see you there!

Useful Projects

  • Perl Data Language - PDL turns Perl into a free, array-oriented, numerical language
  • BioPerl - a community effort to produce Perl code which is useful in biology
  • PerlGSL - new project to provide the Gnu Scientific Library to Perl with a Perlish feel
  • Math::Cephes - a numerical library of mathematical functions
